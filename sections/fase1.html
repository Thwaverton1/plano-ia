Semana 4 (Dias 22–30)

  • Visualizações: histogramas, scatter, heatmap de correlação

Mês 2 (Dias 31–60) — Machine Learning clássico

Semana 5 (Dias 31–37)

Foco: Fundamentos de ML + regressão

  • Conceitos: supervised vs unsupervised, bias-variance tradeoff
  • Regressão linear e logística do zero (sem sklearn primeiro)
  • Métricas: MSE, RMSE, MAE, R², accuracy, precision, recall, F1
  • LeetCode: 6 problemas (começar Hashing e Two Pointers)
Semana 6 (Dias 38–44)

Foco: Sklearn na prática

  • Decision Trees, Random Forest, SVM, KNN
  • Cross-validation: k-fold, stratified k-fold (correto!)
  • Pipelines do sklearn para evitar data leakage

  • LeetCode: 6 problemas (Sliding Window)
Semana 7 (Dias 45–51)

Foco: XGBoost, LightGBM e tuning

  • Gradient boosting: conceito, hiperparâmetros, quando usar
  • Hyperparameter tuning: GridSearch, RandomSearch, Optuna
  • Análise de feature importance
  • LeetCode: 6 problemas (Binary Search)
Semana 8 (Dias 52–60) 1º Projeto

Foco: Construção do 1º projeto real

  • Escolher problema: classificação ou regressão em dataset real
  • Pipeline completo: EDA → feature eng → modelos → avaliação
  • README em inglês explicando o projeto e resultados
  • Subir no GitHub com estrutura profissional de pastas
  • LeetCode: 5 problemas + revisão dos erros
Marco do mês 2: 1º projeto ML completo no GitHub. 36+ problemas LeetCode.
